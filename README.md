## partTimeJob-System


## 使用的技术栈为vue+node.js。

主要实现对兼职广告的发布；

对兼职广告的管理；

对兼职的信息进行修改；

可视化的兼职平台广告数据；

学生用户登录平台的功能和修改个人信息的功能实现。

## 启动项目
```npm i```

```npm run serve```
